# Autonomous Vehicle Driving Agent (CARLA Simulator)

## Project Overview
This repository contains the implementation of an autonomous driving agent developed within the CARLA simulator environment. The primary objective of the agent is to successfully navigate and complete two distinct target routes while ensuring safety, efficiency, and compliance with traffic regulations.

## Key Features & Capabilities
The implemented agent is equipped with advanced decision-making and control logic to handle complex urban and suburban driving scenarios, including:
* **Obstacle Detection & Avoidance:** Dynamically detects static and dynamic obstacles along the path, maintaining safe distances or executing avoidance maneuvers.
* **Vulnerable Road Users (VRUs) Safety:** Full integration of pedestrian and cyclist detection to prevent collisions and yield the right-of-way when necessary.
* **Intersection Management:** Autonomous navigation through complex intersections, managing traffic lights, stop signs, and right-of-way priorities.
* **Dynamic Speed Control:** Adaptive speed management based on road conditions, speed limits, proximity to hazards, and traffic density.

## Evaluation & Routes
The agent was tested and validated across **two different routes** within the CARLA simulator, proving its robustness in diverse environments and its ability to generalize driving behaviors under varying conditions.
